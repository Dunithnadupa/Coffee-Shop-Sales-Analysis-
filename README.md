# ☕ Coffee Shop Sales Data Analysis (Excel)

## 📊 Project Overview
This project analyzes a real-world coffee shop dataset to generate business insights and forecast future sales using Microsoft Excel.

## 🎯 Objectives
- Identify best-selling products
- Analyze peak sales hours and days
- Compare store performance
- Forecast future sales trends

## 🛠 Tools Used
- Microsoft Excel

## 📈 Key Insights
- Coffee products are top revenue drivers
- Peak sales occur during morning hours
- Certain store locations outperform others
- Sales trends can be predicted using forecasting

## 📁 Files Included
- Raw Dataset
- Processed Dataset
- Final PDF Report
- Dashboard & Visualizations

## 💡 Skills Demonstrated
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Pivot Tables
- Data Visualization
- Forecasting
